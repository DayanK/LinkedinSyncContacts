# Extension Icons

Pour le moment, vous devez ajouter 3 icônes ici :

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

**Temporairement**, vous pouvez créer des icônes simples avec ce générateur :
https://www.favicon-generator.org/

Ou utilisez n'importe quelle image LinkedIn / logo de votre choix.

Pour tester l'extension sans icônes, Chrome affichera simplement une icône par défaut.
